Detailed Description

Fornisce un criterio di ordinamento temporale per gli oggetti di tipo Prestito.

Implementa l'interfaccia Comparator per permettere l'ordinamento dei prestiti in base alla data di scadenza (dataFine). È utile per visualizzare in cima alla lista i prestiti più urgenti o già scaduti.
